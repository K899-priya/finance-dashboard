# ✨ Features

   ### 📊 Dashboard
    - Financial summary (Balance, Income, Expense)
    - Interactive balance trend chart
    - Category-based spending chart
    - Insights section with analytics

  ### 💳 Transactions
    - View all transactions
    - Search, filter, and sort
    - Add, edit, delete functionality
    - Category badges for better UI

  ### 👤 Role-Based Access
    - Admin → Full control
    - Viewer → Read-only mode

  ### 🧠 State Management
      - Global state using Zustand

  ### 💾 Data Persistence
      - Stored using localStorage

  ### 🎨 UI/UX
      - Responsive design
      - Dark gradient theme
      - Smooth animations
